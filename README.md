# Инструкция к автотестам

## Установка

npm install
npx playwright install chromium

## Настройка

Скопируйте .env.example в .env и заполните своими данными:
cp .env.example .env

## Запуск

npm test                # headless (без браузера)
npm run test:headed     # с видимым браузером

## Допущения

- Тест написан под Element Web (app.element.io).
- Требуется тестовый аккаунт и комната с правами на запись.
- Локаторы основаны на ARIA-ролях — не ломаются при смене CSS-классов.
- Если публичное окружение нестабильно — рекомендуется локальный
  Element + Synapse через Docker.
